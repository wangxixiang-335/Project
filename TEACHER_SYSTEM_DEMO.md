# 教师管理系统完整演示

## 🎯 系统概述

这是一个完全独立的教师管理系统，与学生系统完全分离。教师账号登录后直接进入教师专用首页，包含完整的功能模块。

## 🚀 快速开始

### 1. 启动服务器
```bash
cd d:\Work\Project
node src/app.js
```

### 2. 访问登录页面
打开浏览器访问：`http://localhost:3000/src/login.html`

### 3. 使用教师账号登录
```

密码: password123
```

登录成功后将自动跳转到教师管理系统首页。

## 📋 功能模块详解

### 🏠 首页（教师专用）

#### 账户信息区域
- **头像显示**：默认显示用户图标，支持更换
- **用户名**：只读显示
- **邮箱**：只读显示  
- **个性签名**：可编辑，支持保存

#### 通知中心
分为三个标签页：

**待审批** 📋
- 显示学生提交的需要当前教师审批的成果
- 条形框布局：封面图 + 成果名称 + 提交时间
- 状态：无标记（待审批状态）
- 操作：右侧清除按钮

**通过** ✅
- 显示已通过的成果通知
- 包括：教师自己成果的通过 + 作为指导老师收到的学生成果通过通知
- 显示：分数信息
- 状态：绿色对勾图标
- 操作：右侧清除按钮

**驳回** ❌
- 显示被驳回的成果通知
- 包括：教师自己成果的驳回 + 作为指导老师收到的学生成果驳回通知  
- 显示：驳回原因
- 状态：红色叉号图标
- 操作：右侧清除按钮

**通知持久化**
- 通知存储在本地，未清除的通知在重新登录后仍然显示
- 支持"全部清除"一键清空所有通知

### 📝 成果审批

#### 搜索筛选
- **班级选择**：下拉选择器
- **类型筛选**：成果类型下拉选择
- **名称搜索**：输入框搜索成果名称
- **姓名搜索**：输入框搜索学生姓名

#### 审批列表
表头显示：成果名称 | 类型 | 姓名 | 指导老师 | 提交时间 | 操作

列表项包含：
- 成果基本信息
- 学生信息
- 指导老师信息
- 提交时间
- "批改"操作按钮

#### 审批操作
点击"批改"进入成果预览界面：
- **预览区域**：展示成果完整内容
- **驳回按钮**：点击弹出驳回原因输入框
- **通过按钮**：点击弹出分数输入框
- **消息推送**：审批结果自动推送给学生和指导老师

### ✍️ 成果发布

#### 编辑/预览切换
- 顶部状态切换按钮：编辑 | 预览
- 预览模式下只显示内容，不可编辑

#### 内容编辑区域

**第一行：基本信息**
- 标题输入框
- 成果类型选择器
- 封面图上传

**第二行：人员信息**  
- 合作伙伴输入
- 指导老师选择

**第三行：富文本编辑器**
- 支持图文混排
- 格式化工具栏
- AI一键布局功能
- AI一键润色功能

**第四行：演示视频**
- 视频链接输入
- 时长限制提示

**第五行：附件管理**
- 加号按钮上传文件
- 文件图标展示
- 支持多个附件
- 个数限制提示

#### 操作按钮
- **存草稿**：确认弹窗 → 保存到草稿箱
- **发布**：选择审批人弹窗 → 提交审批 → 进入审核中状态

### 📊 成果管理

#### 状态筛选栏
- **全部成果**：显示所有状态
- **已发布**：已通过审批的成果
- **审核中**：等待审批的成果
- **未通过**：被驳回的成果
- **草稿箱**：保存的草稿

#### 搜索功能
- 右侧名称搜索框
- 实时筛选显示

#### 成果卡片布局
每个卡片包含：
- **左侧**：封面图
- **中间**：标题 + 发布时间
- **右侧**：操作按钮组

#### 不同状态的操作

**已发布** 📢
- 编辑：修改后需要重新提交审批
- 删除：物理删除

**审核中** ⏳
- 撤回：改为草稿状态，可在草稿箱编辑

**未通过** ❌
- 编辑：修改内容
- 删除：物理删除
- 存草稿：变更为草稿状态
- 发布：重新提交审批

**草稿箱** 📝
- 编辑：继续编辑
- 删除：物理删除

#### AI助手功能
对驳回的成果，AI会根据教师驳回原因生成解决方案建议

### 📚 成果库管理

#### 综合搜索
- **班级选择**：按班级筛选
- **类型选择**：按成果类型筛选
- **分数选择**：按分数区间筛选
- **名称搜索**：搜索成果名称
- **姓名搜索**：搜索学生姓名

#### 数据表格
表头：成果名称 | 分数 | 类型 | 姓名 | 指导老师 | 提交时间 | 操作

- 只读查看模式
- 点击"查看"查看详细内容
- 支持排序和分页

### 📈 数据看板

#### 统计卡片
- **总发布量**：显示成果总数
- **已通过**：显示通过数量
- **待审批**：显示等待审批数量
- **已打回**：显示驳回数量

#### 图表区域

**发布量统计图**
- 按月/周显示发布趋势
- 支持时间范围选择

**班级成绩分布图**
- 选择特定班级
- 显示学生成绩分布
- 支持饼图/柱状图切换

## 📝 使用指南

### 首次登录
1. 访问登录页面
2. 输入教师账号密码
3. 系统自动跳转到教师首页

### 日常使用流程
1. **首页**：查看通知和账户信息
2. **成果审批**：处理学生提交的审批请求
3. **成果发布**：创建和发布新成果
4. **成果管理**：管理自己发布的成果状态
5. **成果库**：查看所有学生成果
6. **数据看板**：查看统计数据

### 常见操作
- **审批流程**：查看 → 预览 → 通过/驳回
- **发布流程**：编辑 → 存草稿/发布 → 选择审批人
- **状态管理**：查看各状态 → 执行相应操作

## 🎯 与学生系统的区别

| 功能 | 教师系统 | 学生系统 |
|------|----------|----------|
| 首页 | 通知中心 + 账户管理 | 成果展示 + 快速操作 |
| 审批功能 | ✅ 可审批学生成果 | ❌ 无审批权限 |
| 发布功能 | ✅ 可发布教师成果 | ✅ 可发布学生成果 |
| 管理功能 | ✅ 全部管理权限 | ✅ 仅管理自己的成果 |
| 数据看板 | ✅ 统计分析功能 | ❌ 无数据看板 |

教师系统是一个功能完整的独立管理系统，专注于教学管理和成果审批工作。
