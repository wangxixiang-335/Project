# 教师管理系统使用指南

## 🎯 系统概述

这是一个完整的教师管理系统，包含以下功能模块：

- **首页** - 账户信息管理和通知中心
- **成果审批** - 学生成果的审批和管理
- **成果发布** - 教师发布自己的成果
- **成果管理** - 教师个人成果的状态管理
- **成果库管理** - 查看所有已发布的成果
- **数据看板** - 统计图表和数据分析

## 📱 功能详解

### 🏠 首页

#### 账户信息管理
- **头像设置**: 点击更换头像按钮上传个人头像
- **用户信息**: 显示用户名和邮箱（只读）
- **个性签名**: 可编辑的个人签名，保存后更新到个人资料

#### 通知中心
通知分为三个标签页：

1. **待审批**
   - 显示学生发布并选择当前教师作为审批人的成果
   - 每条通知显示：封面图、成果名称、提交时间
   - 右侧显示待审批状态图标（⏳）

2. **通过**
   - 显示审批通过的成果通知
   - 包含：教师自己成果的通过、学生成果的通过
   - 右侧显示绿色对勾（✓）

3. **驳回**
   - 显示被驳回的成果通知
   - 显示驳回原因
   - 右侧显示红色叉号（✗）

#### 通知操作
- **清除单条**: 点击通知右上的灰色叉号按钮
- **全部清除**: 点击右上角的"全部清除"按钮
- **持久化**: 未清除的通知在下次登录时仍然显示

### 📋 成果审批

#### 搜索功能
- **班级筛选**: 按班级过滤学生成果
- **类型筛选**: 按成果类型过滤
- **名称搜索**: 搜索成果标题或学生姓名

#### 审批列表
列表包含以下信息：
- 成果名称
- 成果类型
- 学生姓名
- 指导老师
- 提交时间
- 操作按钮（批改）

#### 审批流程
1. 点击"批改"按钮进入成果预览
2. 查看成果详情：标题、学生信息、内容、附件图片
3. 选择审批结果：
   - **通过**: 弹窗输入分数，确认提交
   - **驳回**: 弹窗输入驳回原因，确认提交
4. 系统自动发送通知给学生及指导老师

### ✍️ 成果发布

#### 基本信息
1. **标题**: 成果的标题（必填）
2. **成果类型**: 从下拉菜单选择类型
3. **封面图**: 上传成果封面图片
4. **合作伙伴**: 填写合作伙伴信息
5. **指导老师**: 填写指导老师信息

#### 内容编辑
1. **富文本编辑器**: 支持图文混排
2. **AI辅助功能**:
   - **AI布局**: 自动整理内容结构
   - **AI润色**: 优化文字表达
3. **演示视频**: 添加视频演示链接
4. **附件上传**: 支持多个文件附件

#### 发布选项
1. **存草稿**: 保存到草稿箱，可继续编辑
2. **发布**: 提交审批流程，选择审批人

### 📁 成果管理

#### 状态分类
- **全部成果**: 显示所有状态的项目
- **已发布**: 审批通过的成果
- **审核中**: 正在审批的成果
- **未通过**: 被驳回的成果
- **草稿箱**: 已保存但未发布的草稿

#### 项目操作
不同状态下的操作：

1. **已发布**:
   - 编辑（修改后需重新审批）
   - 删除

2. **审核中**:
   - 撤回（撤回后变为草稿状态）

3. **未通过**:
   - 编辑（可查看驳回原因）
   - 删除
   - 存草稿（修改后保存为草稿）
   - 发布（修改后重新提交审批）

4. **草稿箱**:
   - 编辑（继续编辑内容）
   - 删除（物理删除）

#### 项目卡片显示
- 左侧：封面图
- 右侧上：标题
- 右侧下：发布时间
- 最右上：状态标签
- 底部：操作按钮

### 📚 成果库管理

#### 搜索筛选
- **班级**: 按班级筛选
- **类型**: 按成果类型筛选
- **分数**: 按分数段筛选
- **名称**: 搜索成果名称或学生姓名

#### 列表显示
表格显示以下信息：
- 成果名称
- 分数
- 类型
- 姓名
- 指导老师
- 提交时间
- 操作（查看详情）

### 📊 数据看板

#### 统计图表
1. **发布量统计图**: 显示时间维度的成果发布趋势
2. **班级成绩分布图**: 按班级显示学生成绩分布，可选择不同班级查看

#### 数据概览
四个关键指标卡片：
- 总项目数
- 已通过数量
- 待审批数量
- 已打回数量

## 🐛 故障排除

### 常见问题

1. **登录失败**
   - 检查后端服务是否启动
   - 确认邮箱和密码正确
   - 检查网络连接

2. **数据不显示**
   - 检查API接口响应
   - 确认Token有效性
   - 查看浏览器控制台错误

3. **操作无响应**
   - 检查网络连接
   - 确认后端API正常
   - 刷新页面重试

### 调试方法
1. 打开浏览器开发者工具
2. 查看Console和Network标签
3. 检查API请求和响应
4. 查看本地存储中的Token
