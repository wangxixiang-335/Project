## 🎨 首页功能（教师专用）

### 账户信息区
- **头像显示和更换**
- **用户名和邮箱显示**
- **个性签名编辑**

### 通知中心
**三个标签页：**

#### 📋 待审批
- 学生提交的审批请求
- 显示：封面图 + 成果名称 + 提交时间
- 状态：无标记
- 操作：清除按钮

#### ✅ 通过
- 已通过的成果通知
- 包含：自己成果通过 + 指导学生成果通过
- 显示：分数信息
- 状态：绿色对勾图标
- 操作：清除按钮

#### ❌ 驳回
- 被驳回的成果通知
- 包含：自己成果驳回 + 指导学生成果驳回
- 显示：驳回原因
- 状态：红色叉号图标
- 操作：清除按钮

### 通知持久化
- 本地存储，重新登录仍然显示
- 支持"全部清除"功能

## 📝 成果审批

### 搜索筛选
- 班级选择
- 成果类型选择
- 成果名称搜索
- 学生姓名搜索

### 审批列表
表头：成果名称 | 类型 | 姓名 | 指导老师 | 提交时间 | 操作

### 审批流程
1. 点击"批改"查看详细内容
2. **驳回**：输入原因 → 发送通知给学生和指导老师
3. **通过**：输入分数 → 发送通知给学生和指导老师

## ✍️ 成果发布

### 编辑界面
1. **基本信息**：标题、类型、封面图
2. **人员信息**：合作伙伴、指导老师
3. **富文本编辑器**：图文混排、AI辅助
4. **演示视频**：视频链接、时长限制
5. **附件管理**：多文件上传、图标显示

### 操作选项
- **存草稿**：保存到草稿箱

## 📊 成果管理

### 状态分类
- 全部成果
- 已发布 ✅
- 审核中 ⏳
- 未通过 ❌
- 草稿箱 📝

### 状态操作
| 状态 | 可用操作 | 说明 |
|------|----------|------|
| 已发布 | 编辑、删除 | 编辑后需重新审批 |
| 审核中 | 撤回 | 撤回后变草稿 |
| 未通过 | 编辑、删除、存草稿、发布 | 灵活的状态转换 |
| 草稿箱 | 编辑、删除 | 继续完善或删除 |

### AI助手
对驳回的成果自动生成改进建议

## 🚀 使用流程

### 日常操作
1. **登录系统** → 进入首页查看通知
2. **处理审批** → 审批学生提交的成果
3. **发布成果** → 创建和管理自己的成果
4. **查看数据** → 通过数据看板了解统计信息

### 审批流程
`学生提交` → `教师收到待审批通知` → `查看并审批` → `结果通知` → `数据统计`

### 发布流程
`创建成果` → `编辑内容` → `选择审批人` → `提交审核` → `等待审批` → `通过/驳回` → `发布/修改`

## 📋 故障排除

### 常见问题

**Q: 登录失败？**
A: 检查邮箱密码是否正确，确保服务器正在运行

**Q: 页面无法访问？**
A: 确认服务器已启动，检查端口3000是否被占用

**Q: 通知不显示？**
A: 清除浏览器缓存，检查LocalStorage权限

**Q: 数据不更新？**
A: 刷新页面，检查网络连接和API响应
