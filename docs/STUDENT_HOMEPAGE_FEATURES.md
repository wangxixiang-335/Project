# 学生首页功能说明

## 概述
学生登录后默认进入全新的"首页"界面，提供完整的通知系统和用户资料管理功能。

## 主要功能

### 1. 导航菜单
- **首页**: 展示通知中心和个人资料
- **成果发布**: 提交新的项目成果
- **成果管理**: 查看和管理已提交的项目
- **数据看板**: 查看个人统计数据和项目分析

### 2. 账户信息区域
- **头像设置**: 支持自定义头像URL
- **个性签名**: 可编辑的个人签名
- **用户信息**: 显示用户名和身份

### 3. 通知中心
通知中心分为两个标签页：

#### 通过通知
- 显示教师审批通过的项目
- 展示项目封面图
- 显示项目标题和得分
- 绿色对勾图标表示通过状态
- 支持单独清除或全部清除

#### 驳回通知
- 显示教师驳回的项目
- 展示驳回原因
- 红色叉号图标表示驳回状态
- 支持单独清除或全部清除

### 4. 通知特性
- **持久化**: 通知状态保存在服务器，重新登录后仍然显示
- **实时更新**: 教师审批后立即显示通知
- **批量清除**: 支持一键清除所有通过/驳回通知
- **单条清除**: 每个通知项都有独立的清除按钮

## 技术实现

### 前端组件
- `StudentHomepage.jsx`: 主要首页组件
- `NavigationMenu.jsx`: 导航菜单组件
- `NotificationItem.jsx`: 通知项子组件

### 后端API
- `GET /api/notifications`: 获取用户通知列表
- `DELETE /api/notifications/:id`: 清除单个通知
- `GET /api/notifications/unread-count`: 获取未读通知数量
- `PUT /api/users/profile`: 更新用户资料

### 数据库结构
通知数据基于现有的`projects`表，通过项目状态变化来生成通知：
- `status = 1`: 项目通过
- `status = 2`: 项目驳回

## 使用流程

1. **学生登录**: 使用学生账号登录系统
2. **进入首页**: 自动跳转到首页，查看通知
3. **查看通知**: 在通过/驳回标签页查看项目审批结果
4. **管理资料**: 点击"编辑资料"更新头像和签名
5. **清除通知**: 点击单个清除按钮或"全部清除"按钮
